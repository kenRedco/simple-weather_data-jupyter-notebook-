# Simple Weather Data Analysis

## Overview
This project involves analyzing weather data using Jupyter Notebook. The dataset includes various weather parameters, and the analysis aims to extract insights such as visibility, wind speed, and weather conditions.

## Installation

To run this project, you need to have Python and Jupyter Notebook installed on your machine. You can install the required packages using pip:

```bash
pip install pandas sqlalchemy mysql-connector-python jupyterlab

Analysis
In this project, we perform the following analyses:
Find all records where the weather was exactly clear.
Count the number of times the wind speed was exactly 4 km/h.
Count the number of weather conditions that include snow.
Calculate the mean visibility of the dataset.
